Usted debe utilizar objetos de datos de transferencia, porque DTOs sólo pueden ser transferidos a otro cliente GWT.

Debe crear y ExamDto QuestionDto y después de recibir el examen objeto de MySQL que tenía que convertirlo en ExamDto.

Por el lado de cliente que va a operar sólo DTO. Si desea guardar el examen a MySQL que es necesario convertir ExamDto de examen.

Para convertir a DTOs POJOs y la espalda debe usar Bulldozer .

Para utilizar el bulldozer es necesario asignar DTOs POJOs y con las asignaciones de bulldozer. Yo uso las asignaciones personalizadas a través de archivos XML bulldozer .
